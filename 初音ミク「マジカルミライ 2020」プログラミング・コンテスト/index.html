「TextAlive（テキストアライブ）」とは

TextAliveは、楽曲中の歌声に合わせて歌詞をアニメーションさせるリリックビデオの制作支援サービスです。Web上で公開されている楽曲・歌詞コンテンツを利用して、ブラウザ上でリリックビデオを誰でも簡単に制作できます。

今回新しく公開された「TextAlive App API」は、TextAliveの機能を使って、歌詞が音楽に合わせてタイミングよく動くWebアプリケーションを開発できるJavaScript用のライブラリです。一般的なWebアプリケーション用の開発環境で、必要に応じて three.js, PixiJS, p5.js など好みのライブラリと組み合わせながら、自由な映像表現をプログラミングできます。

TextAlive および TextAlive App API は、産業技術総合研究所が研究開発をして一般公開しています。

応募方法


「応募方法」「応募のきまり」を守っていただける方であればどなたでも応募可能です。


「応募のきまり」はこちら

無料

Webアプリケーションのソースコード（HTML、CSS、JavaScriptなど）

  • HTTPサーバ上に設置するだけで動作する静的アプリケーションのみとします（PHP、Ruby、Node.jsなどを利用した動的なレスポンスを返すサーバアプリケーションは不可）
  • ソースコードのビルド用スクリプト（Gulp, Webpack, Parcelなど一般的なビルドツールを利用したもの）は含めて構いません

GitHub

  • ご自身のGitHubアカウントにプライベートリポジトリを作成し、そのリポジトリに応募するプログラムのソースコードを保存して、本コンテストのGitHubアカウント magicalmirai-proconに共有してください
  • 応募フォームにプライベートリポジトリのURLを記載して送信することで応募完了となります
  • リポジトリは、審査中はプライベートリポジトリとし、審査終了後にのみパブリック化が可能となります。（詳しくは応募のきまりをご参照ください）
  1. 応募締め切り後、主催側により審査で入選作品を選定します
  2. 入選作品をマジカルミライ会場、マジカルミライWebサイト、YouTubeで公開し、ミクナビにてユーザー投票を行います
  3. 投票結果を集計して入賞作品を決定します
  4. マジカルミライ企画展ステージで入賞作品を発表します

※賞の種類は検討中です。賞金や賞品はございません。

FAQ

Q. 応募作品中で、初音ミク等の画像を利用してよいのでしょうか？

A. いわゆる二次創作作品については、「ピアプロキャラクターズ」のみ登場させることができます。これに限らず画像をお使いいただくことは可能ですが、必ず

  • ご自身で制作したもの

  • 正規に利用の許諾を受けたもの

のいずれかをお使いいただき、主催者に対して権利面で問題がないことを保証していただきます。また、正規に利用の許諾を受けた作品をお使いいただく際は、その作品の作者のお名前をアプリ中に明示的かつ合理的な方法で記載するようにしてください。

Q. 応募作品中で、対象楽曲以外の効果音等も鳴らしてよいのでしょうか？

A. 効果音等を鳴らすことは可能ですが、以下の点を遵守していただきます。

  • 必ずご自身で制作したものか、正規に利用の許諾を得た効果音をお使いいただき、主催者に対して権利面で問題がないことを保証していただきます。

  • 効果音が対象楽曲の一部を構成するような印象を与える使用はお控えください。

Q. 応募締め切り後にGitHubのプライベートリポジトリへコミットしてもよいでしょうか？

A. 審査時には応募締め切り時点での最終コミットを採用します。したがって、コミットしても審査には影響しません。
